실시간 공유 DB 연결
팀 공유 DB가 이미 연결되어 있습니다.
링크에 접속한 모든 팀원이 동일한 데이터를 실시간으로 공유합니다. 별도 설정이 필요 없습니다.
아래 정보는 시스템에 미리 설정된 값이며 수정할 필요가 없습니다. 다른 Bin으로 변경하려면 값을 수정 후 [연결] 버튼을 누르세요.
① JSONBin 가입 (무료) → jsonbin.io/signup
② API Key 복사
로그인 → 우측 상단 프로필 → API Keys → X-MASTER-KEY 복사
③ Bin 만들기
좌측 메뉴 Bins → CREATE BIN → 아래 JSON 붙여넣기 → CREATE
{"vehicles":[],"drivers":[],"reservations":[],"logs":[],"expenses":[],"maintenances":[],"seq":{"vehicle":1,"driver":1,"reservation":1,"log":1,"expense":1,"maint":1}} 생성된 URL의 마지막 부분(Bin ID)을 복사
④ 아래 두 칸에 입력 → 한 번만 설정하면 자동 저장됩니다
API Key는 본인 브라우저에만 저장되며 어디에도 전송되지 않습니다. 팀원 4명 각자 한 번씩만 입력하면 됩니다.
